# Student Performance Prediction Dashboard

An interactive Streamlit dashboard for predicting student performance in academic assessments using machine learning models.

## Project Overview

This project addresses three research questions:
- **RQ1**: Predict Midterm I scores using early assignments and quizzes
- **RQ2**: Predict Midterm II scores using Midterm I + intermediate assessments  
- **RQ3**: Predict Final Exam scores using all previous assessments

## Features

- **Interactive Data Exploration**: Visualize dataset statistics and distributions
- **Multiple ML Models**: Compare Linear, Polynomial, and baseline models
- **Bootstrap Analysis**: 95% confidence intervals for model performance
- **Feature Importance**: Understand which factors drive predictions
- **Comprehensive Reporting**: Executive summary and recommendations
